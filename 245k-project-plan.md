# 全球最佳商品百科全书 - 245,317个品类项目规划

## 🎯 **项目愿景**
建立全球最大、最全面的商品评选数据库，为245,317个品类评选出各维度下的最佳商品。

## 📊 **项目规模**

### 基础数据
- **一级分类**: 49个
- **二级分类**: 3,525个
- **三级品类**: 245,317个

### 评选规模
假设每个品类有：
- **价格区间**: 3-5个 (取平均值 x=4)
- **评选维度**: 5-8个 (取平均值 y=6.5)

**最终评选数量**: 
```
245,317 × 4 × 6.5 = 6,378,242 个评选结果
```

### 数据量估算
| 数据类型 | 单条大小 | 总条数 | 总大小 | 说明 |
|----------|----------|--------|--------|------|
| 品类信息 | 2KB | 245,317 | 500MB | 基础信息 |
| 价格区间 | 1KB | 981,268 | 1GB | 每个品类4个区间 |
| 评选维度 | 1KB | 1,594,560 | 1.6GB | 每个品类6.5个维度 |
| 评选结果 | 5KB | 6,378,242 | 32GB | 详细评选理由 |
| **总计** | - | **9,199,387** | **35GB+** | 完整数据库 |

### 数据库设计
```sql
-- 核心表结构
CREATE TABLE categories_245k (
    id INT PRIMARY KEY AUTO_INCREMENT,
    level1 VARCHAR(200) NOT NULL,
    level2 VARCHAR(200) NOT NULL,
    name VARCHAR(500) NOT NULL,
    full_path VARCHAR(1000) UNIQUE NOT NULL,
    status ENUM('pending', 'processing', 'completed', 'failed'),
    INDEX idx_level1 (level1),
    INDEX idx_level2 (level2),
    INDEX idx_status (status)
);

CREATE TABLE price_ranges (
    id INT PRIMARY KEY AUTO_INCREMENT,
    category_id INT NOT NULL,
    level VARCHAR(50) NOT NULL,
    min_price DECIMAL(10,2),
    max_price DECIMAL(10,2),
    description TEXT,
    FOREIGN KEY (category_id) REFERENCES categories_245k(id),
    INDEX idx_category (category_id)
);

CREATE TABLE evaluation_dimensions (
    id INT PRIMARY KEY AUTO_INCREMENT,
    category_id INT NOT NULL,
    name VARCHAR(200) NOT NULL,
    weight DECIMAL(3,2),
    description TEXT,
    FOREIGN KEY (category_id) REFERENCES categories_245k(id),
    INDEX idx_category (category_id)
);

CREATE TABLE best_products (
    id INT PRIMARY KEY AUTO_INCREMENT,
    category_id INT NOT NULL,
    price_range_id INT NOT NULL,
    dimension_id INT NOT NULL,
    brand VARCHAR(200),
    product_name VARCHAR(500),
    reason TEXT,
    confidence INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES categories_245k(id),
    FOREIGN KEY (price_range_id) REFERENCES price_ranges(id),
    FOREIGN KEY (dimension_id) REFERENCES evaluation_dimensions(id),
    UNIQUE KEY uniq_evaluation (category_id, price_range_id, dimension_id)
);
```

```javascript
// 分布式处理架构
const workers = [
    { id: 1, range: '0-50000' },
    { id: 2, range: '50000-100000' },
    { id: 3, range: '100000-150000' },
    { id: 4, range: '150000-200000' },
    { id: 5, range: '200000-245317' }
];

// 每个worker独立处理
workers.forEach(worker => {
    processCategoryRange(worker.range);
});
```
